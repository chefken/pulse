# pulse

A minimalist discipline tracking app built with Flutter.

---

## Overview

**pulse** helps you measure and improve your daily discipline through habit tracking, workout logging, and end-of-day reflection. Designed with a premium, distraction-free UI inspired by Apple and Nothing OS aesthetics.

---

## Features

- **Dashboard** — Discipline score ring, this-week tracker, daily habits and tasks
- **Gym** — Weekly split planner, workout logging (exercises + sets + weight), monthly calendar with workout dots
- **Progress** — 30-day discipline score chart
- **Review** — Daily self-rating with mismatch insight
- **Dark / Light mode** — Smooth animated toggle
- **Offline-first** — All data stored locally via Hive

---

## Tech Stack

| Layer | Choice |
|---|---|
| Framework | Flutter 3.x |
| State management | Provider |
| Local storage | Hive |
| Fonts | Google Fonts (Figtree, DM Sans, Cormorant Garamond) |
| Charts | fl_chart |

---

## Getting Started

```bash
git clone https://github.com/chefken/pulse.git
cd pulse
flutter pub get
flutter pub run build_runner build --delete-conflicting-outputs
flutter run
```
